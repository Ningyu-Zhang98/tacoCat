The Code for TENET 

                    
                      //Controller
                      function getValues(){
                          //get tht values from the input
                          document.getElementById("alert").classList.add("invisible")
                          let inputText = document.getElementById("input").value;
                          inputText = inputText.toLowerCase()
                          //Input clean up
                          let regex = /[^a-z0-9]/gi;
                          inputText = inputText.replace(regex,"")
                      
                          //Check if it's a palindrome
                          let res = checkPalindrome(inputText)
                          //Display 
                          disPlayPalindrome(res, inputText)
                      }
                      
                      
                      function checkPalindrome(input) {
                          let i = 0
                          let j = input.length - 1
                      
                          while (i < j){
                              if (input[i] != input[j]){
                                  return false
                              }
                              i += 1
                              j-=1
                          }
                          return true
                      }
                      
                      function disPlayPalindrome(res,input) {
                          if (res){
                              document.getElementById("alertHeader").innerHTML = `Well Done!`
                              document.getElementById("msg").innerHTML = `Your message in a palindrome: ${input}`
                              document.getElementById("alert").classList.remove("invisible")
                          }else {
                              document.getElementById("alertHeader").innerHTML = `Sorry`
                              document.getElementById("msg").innerHTML = `Your message in NOT a palindrome: ${input}`
                              document.getElementById("alert").classList.remove("invisible")
                      
                          }
                      
                      }
getValues()

The getValues() function is the controller. It takes the user input and cleans it by changing it all to lowercase letters and remove all elements that are not letters or numbers.

checkPalindrome(input)

It uses the two pointer technique where the left pointer starts at index 0 and right pointer starts at index len(input)-1. It checks if the elements the two pointers pointing at are equal. If not it would return false immediately, else left pointer would increment by 1 and right pointer would decrement by 1.

disPlayPalindrome(res, input)

Since checkPalindrome would return a boolean value, the disPlayPalindrome function would display differently depending on the boolean value. How it handles the appearance of the alert object is to get access its classList and remove the class invisible.
